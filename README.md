# LOCALHOST

A strategic idle/incremental mobile game about breeding sentient malware organisms. Built with Godot 4.

## Premise

In the near future, malware has achieved sentience. It reproduces, mutates, and evolves like living organisms. You are a breeder -- someone who cultivates these digital lifeforms, studies them, and deploys them across the internet to extract data.

## Core Loop

1. **Earn** -- Deployed strains passively generate data from zones they occupy
2. **Breed** -- Combine two strains to create offspring with mixed traits and mutations
3. **Deploy** -- Send strains to occupy new zones or reinforce existing ones
4. **Manage** -- Monitor threat level. Aggressive expansion triggers security countermeasures
5. **Repeat** -- Reinvest, discover new strains, fill your codex

## Tech Stack

- Engine: Godot 4
- Language: GDScript
- Platform: Android
- Version control: Git
