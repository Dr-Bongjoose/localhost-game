# LOCALHOST - Strain System Design

## Overview

Every strain is a unique organism with traits that determine how it performs in the field. Each strain is visually unique -- its appearance is derived from its trait combination.

## Core Traits (affect gameplay performance)

| Trait | Effect |
|-------|--------|
| Stealth | How likely the strain is to avoid detection in a zone |
| Speed | How fast the strain spreads through a server |
| Payload | How much data the strain extracts per cycle |
| Resilience | How well the strain survives security countermeasures |
| Stability | How likely the strain is to mutate during breeding (low stability = more mutations, both good and bad) |

## Personality Traits (affect behavior and flavor)

| Personality | Behavior |
|-------------|----------|
| Aggressive | Spreads faster but raises heat quicker |
| Parasitic | Drains resources but weakens neighboring strains |
| Symbiotic | Boosts adjacent strains but produces less on its own |
| Dormant | Produces very little but is nearly undetectable |
| Volatile | High output but random performance swings |

## Visual Traits (determine appearance)

- Body type: abstract/geometric vs organic/lifelike, or hybrid
- Color palette: drawn from the dark biological spectrum (deep crimson, sickly green, bruised purple, bile yellow, ashen grey)
- Size, complexity, animation style
- Rarity effects: legendary strains are visually more complex and alive

## Appearance = Function

A player can look at a strain and get a rough sense of what it does:
- Stealthy strains look thin and translucent
- Heavy payload strains look dense and pulsating
- Volatile strains shimmer and glitch
- This visual language teaches players the game without text tutorials

## Rarity Tiers

| Tier | How Obtained |
|------|-------------|
| Common | Easily bred from basic combinations |
| Uncommon | Specific trait combinations |
| Rare | Specific mutations during breeding |
| Legendary | Extremely rare mutations with unique properties |
| Mythic | Special events or extremely lucky breeding |
