# LOCALHOST - Player Profile & Learning Context

## About the Developer

- Beginner game developer -- has never finished a game
- Has started but not completed a few game development courses
- Programming experience: beginner level in C/C++, Python, JavaScript
- First programming language was ActionScript (Macromedia Flash era, learned accidentally)
- Understands programming concepts but struggles with writing code from scratch
- Wants to LEARN to code through this project, not just have a game built for them
- Goal: after making enough games together, be able to write own code and build independently

## Learning Approach

- I write code with heavy comments explaining WHY each piece exists
- Each time we add a feature, explain the specific Godot concepts needed
- Contextual learning sticks better than abstract tutorials
- Give small challenges to try writing code themselves
- User learns by modifying code I write -- change numbers, tweak traits, adjust balance
- GDScript will feel like Python to them (confirmed: they can read GDScript and roughly understand it)

## Communication Style

- User is new to AI agents and agent frameworks
